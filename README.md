# Required 
Php ^8.1

Composer

Mysql DB Created

# Instructions
1 clone the repository

2 composer install

3 create .env file from .env.exemplo

4 php artisan key:generate

5 setup the env variables

  DB_HOST=
  
  DB_PORT=
  
  DB_DATABASE=
  
  DB_USERNAME=
  
  DB_PASSWORD=
  

6 php artisan serve
